# 🎤 AI Interviewer — Mock Interview Platform

AI-powered mock interview platform with adaptive questions, real-time answer analysis, and detailed feedback.

## Features
- Adaptive AI questions based on your answers
- Real-time scoring and feedback per answer
- General/Behavioral and Technical (AI & Automation) interview types
- Session history saved to database
- Overall readiness score and improvement tips

## Built With
- Next.js 14
- Groq API (Llama 3.3 70B)
- Supabase (PostgreSQL)
- Tailwind CSS
